## Как пользоваться сервисом

Приложение представляет собой консольное приложение для управления личными финансами. Вот основные шаги, чтобы начать работу:

### Запуск программы:

Откройте файл Main.java и запустите его. Он является точкой входа в приложение.

### Регистрация/Авторизация:

При первом запуске программа предложит авторизоваться или зарегистрироваться. Введите логин и пароль.
После успешной регистрации данные сохраняются в системе.

### Доступные действия:

После входа в аккаунт пользователь может выполнять команды, такие как:
Добавление доходов и расходов.
Управление категориями бюджета.
Просмотр состояния кошелька и бюджета.

### Сохранение данных:

При выходе из системы данные пользователя сохраняются в файл, чтобы быть доступны при следующем входе.

## Какие команды поддерживаются
### Приложение поддерживает следующие команды:

### Работа с финансами:

Добавление доходов.
Добавление расходов.
Просмотр общей суммы доходов и расходов.
Работа с категориями:

Создание и управление категориями.
Установление бюджета для каждой категории.
Отображение информации:

Просмотр текущего состояния бюджета.
Отображение оставшихся лимитов для категорий.

### Авторизация/Регистрация:

Регистрация нового пользователя.
Авторизация существующего пользователя.

### Сохранение и загрузка:

Данные автоматически сохраняются при выходе из программы.
Загрузка данных при входе.

## Как протестировать код
### Запустите программу и попробуйте:
Зарегистрироваться.
Добавить несколько операций.
Выйти и перезапустить, проверяя, что данные сохраняются.
